### git
```
1.到目錄
2.git init 初始化
3.git remote add origin https://xxxx
(git remote -v 查看遠端位置)
4.git add . //將檔案交給git 
5.git commit -m '' //說明
6.git push -u origin master //第一次上傳到雲端
7.Source Tree 創建 git flow
8.add feature
```
